# Inti

API Integration APP for BL and Shopify.

The project structure:

|- server.ts
|- src/
|!|- app.js
| |- controllers/
| |- services/
| |- routes/
| |- middleware/
| |- models/
| |- config/
| |- utils/
| |- tests/

The project uses https://github.com/MONEI/Shopify-api-node as Shopify Admin API connector.
